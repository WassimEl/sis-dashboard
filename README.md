# 🎮 Siskath Dashboard

Dashboard interactif pour streameuse avec compagnon virtuel, suivi de runs et streams.

## ✨ Fonctionnalités

- 🏃 **Run Logger** - Suivi des courses (distance, durée, notes)
- 🎮 **Stream Tracker** - Suivi des streams (jeux, durée, viewers)
- 🎲 **Jeu Aléatoire** - Tirage de jeux via l'API RAWGG
- 🎬 **Dashboard Live** - Roue de la fortune personnalisable pour le chat
- 🏆 **Défis Hebdomadaires** - Objectifs avec XP bonus
- 🇯🇵 **Apprentissage Japonais** - Mots en Hiragana/Katakana/Kanji
- 🤖 **IA Coach** - Conseils personnalisés sans API
